## Project Overview

DeepTutor is the comprehensive creation of YuvisTechPoint. This project represents a complete agent-native intelligent tutoring platform developed from the ground up, including:

- **Architecture Design** — Two-layer plugin model (Tools + Capabilities)
- **Core Engine** — Agent-native orchestration system
- **Features** — Chat, Deep Solve, Quiz Generation, Deep Research, Math Animator, Visualize
- **Backend** — Python-based FastAPI server with LLM integration
- **Mobile** — Flutter Android client (`deeptutor_mobile/`)
- **Deployment** — Docker, Vercel, and multi-user support
- **Documentation** — Complete guides and CLI reference
